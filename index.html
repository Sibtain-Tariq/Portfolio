Automated Business Software for Sales Agents

This is an ongoing project where I am building an automated business management software tailored for a local bakery supply business. The focus is on simplifying sales agent operations by centralizing and automating data entry. The system is designed so that the user only needs to input data on a single page, and all related calculations, reports, and summaries are updated automatically across the application.

The software includes modules for delivery challans, agent-wise financial summaries, daily sale tracking, discount handling, and net sale calculations. Each page is designed with ease of use in mind, while ensuring accurate, real-time calculations for sales performance, commissions, discounts, and payments.

Although still in development, we plan to integrate Power BI dashboards for better visual reporting and an AI chatbot to provide on-page assistance and real-time support. This project has helped me combine my technical skills in React and data management with real business requirements to build something practical and scalable for local industries.

Ethereum Price Forecasting using ARIMA and ARIMAX

In this project, I built a forecasting system for Ethereum (ETH/USDT) using time series analysis with ARIMA and ARIMAX models. The dataset was collected from Yahoo Finance and included daily open, high, low, close, and volume values. I performed data cleaning, feature selection, and implemented an ARIMAX model using Python, integrating exogenous variables such as market volume and price fluctuations. I tested multiple (p, d, q) configurations, tracked model performance using RMSE and MAPE, and selected the best model to generate accurate 30-day price forecasts. Visualizations were created to compare real vs predicted values and highlight market trends. This project helped me strengthen my understanding of time series forecasting, model tuning, and financial data analysis.
